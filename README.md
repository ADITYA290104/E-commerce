🛒 E-Commerce Frontend

A modern and responsive E-commerce frontend application where users can browse products, view details, and manage their shopping cart.

🚀 Features
🏠 Home page with product listings
📄 Product details page
🔢 Update product quantity
📱 Fully responsive design
⚡ Fast performance using Vite
🛠️ Tech Stack
React.js
Vite
JavaScript (ES6+)
CSS / Tailwind CSS /
Html
